Mi setup

Cargando núcleo creativo

BenQ MOBIUZ EX240
Monitor para edición y revisión visual

iPad Pro M2
Bocetos, segunda pantalla y flujo creativo

Pebble Keys
Perfil bajo, compacto y limpio

DJI Mic Mini
Audio limpio para grabar contenido

MX Vertical
Ergonomía para sesiones largas

Xiaomi Lightbar
Luz precisa sin reflejos

Tengo un BenQ MOBIUZ EX240 y el núcleo es una Mac Mini M4, que ofrece la potencia necesaria en un formato compacto. Complemento mi flujo creativo con un iPad Pro M2 como segunda pantalla, la ergonomía del MX Vertical y el Pebble Keys 2 K380s para sesiones largas. En el audio, uso el DJI Mic Mini para capturar un sonido claro y, para una iluminación limpia, la Xiaomi Lightbar.
